## F014 - Lỗi sử dụng Function Module Front-end cũ `WS_DOWNLOAD`

* **ID:** `F014`
* **Nội dung lỗi (エラー内容):** Sử dụng Function Module `WS_DOWNLOAD` bị lỗi do đây là FM phụ thuộc frontend (SAP GUI) cũ, không còn được hỗ trợ.
* **Cách khắc phục (対応方法):** Thay thế bằng Function Module `GUI_DOWNLOAD`. Chú ý tham số `FILENAME` của `GUI_DOWNLOAD` bắt buộc phải truyền vào biến kiểu dữ liệu `STRING` (cần khai báo biến trung gian kiểu `STRING` và gán tên file trước khi truyền).

### Cú pháp cũ (旧)

```abap
CALL FUNCTION 'WS_DOWNLOAD'
  EXPORTING
    FILENAME            = IN_FILENAME " IN_FILENAME có thể là kiểu CHAR hoặc kiểu cũ
    FILETYPE            = 'ASC'
  TABLES
    DATA_TAB            = IN_DATA_TAB
  EXCEPTIONS
    FILE_OPEN_ERROR     = 1
    FILE_WRITE_ERROR    = 2
    INVALID_FILESIZE    = 3
    INVALID_TABLE_WIDTH = 4
    INVALID_TYPE        = 5
    NO_BATCH            = 6
    UNKNOWN_ERROR       = 7
    OTHERS              = 8.
```

### Cú pháp mới (新)

```abap
DATA: PNY_FILE_NAME1 TYPE STRING.
PNY_FILE_NAME1 = IN_FILENAME. " Gán sang biến kiểu STRING bắt buộc cho GUI_DOWNLOAD

CALL FUNCTION 'GUI_DOWNLOAD'
  EXPORTING
    FILENAME            = PNY_FILE_NAME1
    FILETYPE            = 'ASC'
  TABLES
    DATA_TAB            = IN_DATA_TAB
  EXCEPTIONS
    FILE_WRITE_ERROR    = 2
    NO_BATCH            = 6
    INVALID_TYPE        = 5
    UNKNOWN_ERROR       = 7
    OTHERS              = 8.
```
