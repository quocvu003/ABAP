## 1.2. B014 - Lỗi Batch Input hiển thị/tra cứu nhập xuất kho (MB03)

* **ID:** `B014`
* **Nội dung lỗi (エラー内容):** Lệnh gọi Batch Input cho giao dịch hiển thị/tra cứu chứng từ nhập xuất kho (`MB03`) không còn được hỗ trợ trên S/4HANA. Nếu sử dụng `MB03` làm chức năng hiển thị/tra cứu, phải lấy dữ liệu bằng phương pháp khác.
* **Cách khắc phục (対応方法):**
  * Cách 1: Sử dụng BAPI `BAPI_GOODSMVT_GETDETAIL` để lấy chi tiết chứng từ.
  * Cách 2: Truy vấn trực tiếp từ bảng cơ sở dữ liệu `MATDOC`.
  Sau khi lấy được dữ liệu bằng 1 trong 2 cách trên, thực hiện hiển thị dữ liệu kết quả lên Grid bằng ALV (`ALV表示`). *Cần kiểm tra trước với thiết kế (事前確認) xem sử dụng cách nào.*

### Cú pháp cũ (旧)

```abap
CALL TRANSACTION 'MB03' USING TB_BDC
                        MODE 'S'
                        UPDATE 'N'
                        MESSAGES INTO TB_MESSAGE.
```

### Cú pháp mới (新)

**Cách 1: Dùng BAPI lấy chi tiết**

```abap
CALL FUNCTION 'BAPI_GOODSMVT_GETDETAIL'
  EXPORTING
    MATERIALDOCUMENT = LV_MBLNR
    MATDOCUMENTYEAR  = LV_MJAHR
  TABLES
    GOODSMVT_ITEM    = LT_ITEM
    RETURN           = LT_RETURN.
* (Sau đó thực hiện hiển thị danh sách LT_ITEM qua ALV)
```

**Cách 2: Truy vấn trực tiếp từ bảng MATDOC**

```abap
SELECT *
  FROM MATDOC
  WHERE MBLNR = @LV_MBLNR
    AND MJAHR = @LV_MJAHR
  INTO TABLE @DATA(LT_MATDOC).
* (Sau đó thực hiện hiển thị danh sách LT_MATDOC qua ALV)
```
