## 1.3. B016 - Giao dịch MBRL khuyến nghị sử dụng BAPI (Trả lại hàng)

* **ID:** `B016`
* **Nội dung lỗi (エラー内容):** Khuyến nghị sử dụng BAPI thay thế cho việc gọi Batch Input giao dịch Trả lại hàng (`MBRL`). Tùy thuộc vào kịch bản Batch Input của `MBRL`, trường hợp chỉ hủy Nhập kho (GR) là phổ biến nhất. Nếu cần hủy cả Hóa đơn (IV/LIV), cần thực hiện theo quy trình tương ứng.
* **Cách khắc phục (対応方法):**
  * **Trường hợp 1 (Hủy Hóa đơn - LIV):** Gọi BAPI `BAPI_INCOMINGINVOICE_CANCEL`.
  * **Trường hợp 2 (Hủy Nhập kho - GR):** Gọi BAPI `BAPI_GOODSMVT_CANCEL`.
  * **Trường hợp 3 (Đăng ký lại nếu cần):** Nếu quy trình cũ có bước tạo lại thông tin mới sau khi hủy, sử dụng `BAPI_GOODSMVT_CREATE` và `BAPI_INCOMINGINVOICE_CREATE`.

### Cú pháp cũ (旧)

```abap
CALL TRANSACTION 'MBRL' USING TB_BDC
                        MODE 'S'
                        UPDATE 'N'
                        MESSAGES INTO TB_MESSAGE.
```

### Cú pháp mới (新)

**Bước 1: Hủy Hóa đơn (Invoice Cancellation - LIV)**

```abap
DATA: LV_INV_DOC TYPE RE_BELNR,  " Số chứng từ hóa đơn
      LV_FISC_YR TYPE GJAHR,     " Năm tài khóa
      LT_RETURN  TYPE TABLE OF BAPIRET2.

CALL FUNCTION 'BAPI_INCOMINGINVOICE_CANCEL'
  EXPORTING
    INVOICEDOCNUMBER = LV_INV_DOC
    FISCALYEAR       = LV_FISC_YR
  TABLES
    RETURN            = LT_RETURN.

IF LINE_EXISTS( LT_RETURN[ TYPE = 'E' ] ).  " Gặp lỗi thì dừng xử lý
  EXIT.
ENDIF.

CALL FUNCTION 'BAPI_TRANSACTION_COMMIT'
  EXPORTING
    WAIT = ABAP_TRUE.
```

**Bước 2: Hủy Nhập kho (Goods Receipt Cancellation - GR)**

```abap
DATA: LV_MBLNR   TYPE MBLNR,
      LV_MJAHR   TYPE MJAHR,
      LT_RETURN  TYPE TABLE OF BAPIRET2.

CALL FUNCTION 'BAPI_GOODSMVT_CANCEL'
  EXPORTING
    MATERIALDOCUMENT = LV_MBLNR
    MATDOCUMENTYEAR  = LV_MJAHR
  TABLES
    RETURN           = LT_RETURN.

IF LINE_EXISTS( LT_RETURN[ TYPE = 'E' ] ).  " Gặp lỗi thì dừng xử lý
  EXIT.
ENDIF.

CALL FUNCTION 'BAPI_TRANSACTION_COMMIT'
  EXPORTING
    WAIT = ABAP_TRUE.
```

**Bước 3: Tạo lại chứng từ mới (nếu luồng nghiệp vụ cũ yêu cầu tạo lại sau khi hủy)**

```abap
* Tạo lại GR mới
CALL FUNCTION 'BAPI_GOODSMVT_CREATE'
  EXPORTING
    GOODSMVT_HEADER = LS_HEADER
    GOODSMVT_CODE   = LS_CODE   " Ví dụ: '01', '04'
  TABLES
    GOODSMVT_ITEM   = LT_ITEM
    RETURN          = LT_RETURN.

* Tạo lại Hóa đơn (LIV) mới
CALL FUNCTION 'BAPI_INCOMINGINVOICE_CREATE'
  EXPORTING
    HEADERDATA      = LS_HEADERDATA
  TABLES
    ITEMDATA        = LT_ITEMDATA
    RETURN          = LT_RETURN.
```
